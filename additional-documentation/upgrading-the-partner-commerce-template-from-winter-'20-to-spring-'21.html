Updating The Libraries

You must update the following packages to the latest version:

  • @apttus/core ^1903.x.x --> ^2101.x.x
  • @apttus/ecommerce ^1903.x.x --> ^2101.x.x
  • @apttus/elements ^1903.x.x --> ~2101.x.x

After updating the packages, verify if your package.json under dependencies reflects the latest version as under:

    ...
    "@apttus/core": "^2101.x.x",
    "@apttus/ecommerce": "^2101.x.x",
    "@apttus/elements": "^2101.x.x",
    ...

Updating Environment File With Pricing Flag

In the environment file for template, add "pricingFallback" flag and set its default value as False. With this flag, you can determine whether CPQ pricing must be triggered on turbo pricing failure or prompt the user to retry pricing.
